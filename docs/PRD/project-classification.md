# Project Classification

**Technical Type:** Web Application  
**Domain:** General  
**Complexity:** Low

**技术栈：**
- **前端：** Next.js + Tailwind CSS + 组件库（shadcn/ui 或 Headless UI）
- **后端：** Node.js
- **数据库：** PostgreSQL
- **部署：** Vercel
- **存储：** 本地存储（抽象层设计）
- **认证：** OAuth + JWT
- **编辑器：** Tiptap

**项目类型：** 个人博客平台 + 实验性学习项目
