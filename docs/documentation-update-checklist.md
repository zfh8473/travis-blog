# 文档更新检查清单

**用途：** 代码变更时快速检查是否需要更新文档  
**使用时机：** 代码提交前、代码审查时

---

## 🔍 快速检查

### 代码变更类型检查

**如果您的变更涉及以下内容，请检查相应文档：**

- [ ] **技术栈变更** → 检查 PRD、相关 Epic
- [ ] **架构模式变更** → 检查架构文档、相关 Epic
- [ ] **新增功能** → 检查相关 Epic、Story
- [ ] **功能变更** → 检查相关 Epic、Story
- [ ] **API 变更** → 检查 API 文档、相关 Epic
- [ ] **UI/UX 变更** → 检查用户文档、相关 Epic

---

## 📋 详细检查清单

### 技术栈变更

**触发条件：**
- 新增主要依赖库
- 替换主要依赖库
- 框架版本重大升级

**需要更新的文档：**
- [ ] `docs/PRD.md` - 技术栈部分
- [ ] 相关 Epic 文档 - 技术说明部分
- [ ] `docs/architecture.md` - 技术栈说明
- [ ] `package.json` 注释（如需要）

**检查项：**
- [ ] 技术栈名称准确
- [ ] 版本号正确（如需要）
- [ ] 变更原因已记录
- [ ] 其他文档中的引用已更新

---

### 架构模式变更

**触发条件：**
- API Routes → Server Actions
- Client Components → Server Components
- 数据获取方式变更

**需要更新的文档：**
- [ ] 相关 Epic 文档 - 技术说明部分
- [ ] `docs/architecture.md` - 架构模式说明
- [ ] 相关 Story 文档 - 实现细节

**检查项：**
- [ ] 架构模式说明准确
- [ ] 实现方式描述正确
- [ ] 变更原因已记录
- [ ] 优势说明已添加

---

### 功能变更

**触发条件：**
- 新增功能
- 功能行为变更
- 功能移除

**需要更新的文档：**
- [ ] 相关 Epic 文档 - Story 描述
- [ ] 相关 Story 文档 - 接受标准
- [ ] `docs/PRD.md` - 功能需求部分（如需要）
- [ ] 用户文档（如需要）

**检查项：**
- [ ] 功能描述准确
- [ ] 接受标准完整
- [ ] 技术说明正确
- [ ] 用户文档已更新（如需要）

---

### 编辑器/工具变更

**触发条件：**
- 编辑器替换（如 Tiptap → MarkdownEditor）
- 工具库变更

**需要更新的文档：**
- [ ] `docs/PRD.md` - 技术栈部分
- [ ] 相关 Epic 文档 - Story 3.2
- [ ] 相关 Story 文档 - 技术说明

**检查项：**
- [ ] 编辑器名称准确
- [ ] 功能描述正确
- [ ] 变更原因已记录
- [ ] 所有引用已更新

---

## 🚨 常见遗漏场景

### 容易被忽略的更新

1. **Epic 总结部分**
   - 更新完成的故事描述
   - 更新技术成就列表
   - 更新用户体验部分

2. **交叉引用**
   - 其他文档中的技术栈引用
   - 其他文档中的功能引用
   - 其他文档中的架构引用

3. **代码注释**
   - JSDoc 注释中的技术栈说明
   - README 文件中的技术栈说明

---

## ✅ 提交前检查

### 代码提交前必查

- [ ] 已识别所有需要更新的文档
- [ ] 已更新所有相关文档
- [ ] 已检查文档一致性
- [ ] 已记录变更原因（如需要）
- [ ] 已更新文档索引（如需要）

### Git 提交信息建议

```bash
# 好的提交信息示例
docs: 更新 Epic 3.2 反映 MarkdownEditor 实现
docs: 添加 Server Actions 说明到 Epic 3.3
docs: 更新 PRD 技术栈部分

# 包含变更原因
docs: 更新编辑器说明（Tiptap → MarkdownEditor，改进用户体验）
```

## 📞 需要帮助？

**不确定是否需要更新文档？**

1. 检查本文档的检查清单
2. 咨询 PM 或 Architect
3. 在代码审查时提出疑问

**文档更新优先级不确定？**

- **P0（必须）** - 技术栈、架构、核心功能变更
- **P1（建议）** - 功能增强、次要功能变更
- **P2（可选）** - 文档完善、格式优化
