## 🔍 问题排查

如果代码块仍然显示为普通文本，可能的原因：

1. **旧文章内容问题**：
   - 旧文章在转换前已存在，需要重新保存
   - 建议：创建新文章测试

2. **编辑器转换问题**：
   - `markdownToHtml` 函数可能有问题
   - 建议：检查保存到数据库的 HTML 内容

3. **样式应用问题**：
   - Tailwind Typography 插件可能未正确加载
   - 建议：检查浏览器开发者工具中的样式
